# Kernel Bot
Bot pour Kernel

Actual version: First stable version !

## Instructions
* Made for Python 3.6.
* Deps: discord-py

Set-up a virtualenv, `pip install -r requirements.txt` and boom, done.

## Code style

* PEP-8 is required.
* Type hinting is always the best way !
* No restrictions over name, everything is up to your imagination.
